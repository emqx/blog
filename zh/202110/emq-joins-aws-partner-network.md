近日，[EMQ 映云科技](https://www.emqx.com/zh/about)正式加入 AWS 合作伙伴网络 (AWS Partner Network，以下简称 APN) ，成为 AWS（亚马逊云科技）**官方认证合作伙伴**。

APN 是 AWS 的一项全球性合作伙伴计划。加入 APN 意味着 EMQ 通过了 AWS 在技术与架构方面的严格审核。EMQ 提供的 MQTT 服务满足 AWS 的最佳实践要求，可以在可用性、安全性等方面为客户带来有效保障。

未来 EMQ 将借助该计划在技术、业务和市场推广方面的项目化支持，通过优质产品和领先技术，与 AWS 共同帮助企业客户加速云计算之旅。

客户现可通过 AWS 官网 [Partner 页面](https://partners.amazonaws.com/partners/0010h00001jE6qHAAS/EMQ)查询到 EMQ 相关信息。

## EMQ & AWS，强强再合作

作为一家物联网数据基础设施软件供应商，EMQ 为全球上万用户提供了[云原生分布式 MQTT 消息服务器 EMQ X](https://www.emqx.com/zh/products/emqx)，为高可靠、高性能的物联网实时数据移动、处理和集成提供动力，助力企业快速构建关键业务的 IoT 平台与应用。

[EMQ X Cloud](https://www.emqx.com/zh/cloud) 是基于 EMQ X 的全托管 SaaS 服务，免除用户的运维负担，充分利用云计算的规模效应给客户带来高性价比的服务。

发布伊始，EMQ X Cloud 便完成了与 AWS 公有云服务的对接。借助 AWS 遍布全球的稳定基础设施网络和 EMQ X Cloud 海量连接、高并发的产品特性，用户可以在国内和海外多个 AWS 支持的地区部署 MQTT 服务，快速开启物联网业务。

为积极响应“碳中和”战略，今年，EMQ 还与 AWS、SAP 合作打造了面向全产业链的碳排放数字化平台，借助数字化与物联网技术为企业与监管部门提供从生产制造到碳排量、碳足迹分析的一站式服务，帮助企业进行能耗排放监控管理和产业转型升级。

![EMQ+AWS+SAP 碳排放数字化平台架构示意图](https://static.emqx.net/images/5f76cd2ff3592d9da38f9af73928a07c.png)

<center>EMQ+AWS+SAP 碳排放数字化平台架构示意图</center>


在这一绿色解决方案中，EMQ 接入层与网关层主要进行设备端的整体能耗与碳排放数据采集和预处理，并传输至云端，AWS 基于底层传感器架构的分析预测层 & 存储层则负责后续的数据存储、统计、分析与预测，共同为企业提供可靠的碳中和数据支撑。

该方案于今年七月亮相 2021 亚马逊云科技中国峰会上海站，受到了现场观众的高度关注。


## 加速云时代，预见未来

截至目前，90% 以上的全球财富 100 强企业都在使用 AWS 合作伙伴提供的解决方案和服务。EMQ 的加入将为这些企业的物联网创新业务带来更多可能。

一个是全球应用最广泛的云平台，一个是同类型产品全球排名前列的物联网消息服务器，AWS 与 EMQ 的强强联合必将成为云原生时代下物联网领域企业降低成本、提高敏捷性、加速业务创新的最佳选择。

EMQ 将与 AWS 持续探索，让更多企业享受云计算带来的便利，共同服务未来产业与社会。
